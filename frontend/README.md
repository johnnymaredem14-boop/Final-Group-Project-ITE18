# Employee Leave Management System Frontend

Vue.js + Pinia + Vue Router frontend 

## Updated Features
- Role-based access
- Admin CRUD for Employees, Departments, and Leave Types
- Employee leave request CRUD
- Password field when adding employee accounts
- Real-time UI updates through Pinia reactivity
- Data persistence through localStorage

## Run

```bash
npm install
npm run dev
```
